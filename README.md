## Install

```bash
git clone --depth 1 --single-branch https://github.com/w-lianfu/electron-up.git
cd electron-up
yarn
```

## Starting Development

Start the app in the `dev` environment:

```bash
yarn start
```

## Packaging for Production

```bash
yarn package
```
